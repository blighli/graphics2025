# 第二次作业 - 绘制太阳系
使用glad，GLFW，glm。
在hw3目录内使用make或mingw32-make命令编译，输出结果为output/main.exe。
- 实现了放置和穿梭传送门的功能，简单的AABB轴对齐碰撞盒功能。
- wasd移动，鼠标左右键用于改变两个传送门的位置。
- resultx.png等文件为结果预览。
    - result1为成对传送门展示。
    - result2为简单场景展示。
    - result3为传送门递归渲染展示。
- 传送门制作思路参考[该仓库](https://learnopengl-cn.github.io/)。
- OpenGL导入模型，阴影绘制，碰撞体积等功能参考[LearnOpenGL CN](https://learnopengl-cn.github.io/)。
