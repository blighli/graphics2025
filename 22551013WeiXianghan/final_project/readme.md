# 赛车游戏程序

## 项目简介

本项目是一个基于 OpenGL 的赛车游戏程序，使用 C++ 编写，结合了 GLAD、GLFW、GLM 和 LearnOpenGL 提供的工具库。程序实现了一个简单的 3D 场景渲染，包括汽车、赛道、STOP 标志、天空盒等模型，并支持光照、阴影、相机切换等功能。

---

## 功能概述

### 1. **场景渲染**
- **汽车模型**：加载并渲染汽车模型，支持动态移动和旋转。
- **赛道模型**：加载并渲染赛道模型。
- **STOP 标志**：加载并渲染 STOP 标志模型。
- **天空盒**：通过天空盒渲染，提供逼真的环境背景。
参考资料：https://learnopengl-cn.github.io/04%20Advanced%20OpenGL/06%20Cubemaps/#_3

### 2. **光照与阴影**
- 实现了基于光源的阴影渲染，使用正交投影矩阵生成深度图。
- 支持动态光源位置更新。

### 3. **相机控制**
- **自由视角**：用户可通过键盘和鼠标自由控制相机位置和方向。
- **固定视角**：相机固定在汽车后方，随汽车移动和旋转。
- 支持视角切换（按键 `C`）。

### 4. **键盘与鼠标交互**
- **汽车控制**：
  - `↑` 和 `↓`：控制汽车加速和减速
   - 汽车初速度为0，请按下`↑`开始游戏
   - 每按下一次`↑` 和 `↓`，汽车速度+1/-1。
  - `←` 和 `→`：控制汽车左右转向。
- **相机控制**：
  - 相机默认跟随汽车运动，实现赛车游戏中的沉浸效果。
  - 自由视角：`W`、`A`、`S`、`D`、`Space`、`Ctrl` 控制相机移动。
  - 固定视角：`A` 和 `D` 控制相机左右偏移。
- **渲染模式切换**：
  - 按键 `X`：切换线框模式和正常渲染模式。
- **退出程序**：按键 `Esc`。

### 5. **小地图**
- 在窗口左上角渲染一个顶视角，实时显示汽车和赛道的俯视图。
- 可以按 `P` 键开启、关闭该小视图。

## 鸣谢

本项目基于https://github.com/ceynri/3d-racing-games-demo ，在原项目只能通过`↑` 和 `↓`匀速行车的基础上，修改了车辆控制逻辑，增加了车速管理功能，支持加速、刹车、倒车，使其更符合赛车游戏的体感效果。同时，增加了小窗俯视图显示，使游戏功能更加完备。
